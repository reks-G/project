### project 

# 1.
1) name=TaskControl 
2) platformц=telegram

3) Для чего этот проект - Таск Менеджер для компаний
4) Для кого проект - Для компаний
5) Основной сценарий использования - Руководитель создаёт компанию в телеграм боте, куда он может добавить своих сотрудников. Создаётся задача, прописанно т/з задачи и deadline, работники после выполнения задачи помечают ее выполненной.
6) MVP - Операции с задачами(создание, удаление, изменение), назначение кураторов над задачей(кто будет следить за выполнением), Система ролей(Тоесть в компании может быть несколько ролей, настраивается индивидуально для каждого), Дедлайны с контролем времени, Фильтрация задач по deadline'у и по важности.
